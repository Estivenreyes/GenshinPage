About the redemption code

  1. Before redeeming a code, make sure you have started an in-game game and linked your game to your miHoYo Pass. Otherwise, you will not be able to redeem the code.

  2. After redeeming a code, you will receive the redeemed items via in-game mail. Check your email to verify that you received them correctly.

  3. Pay attention to the redemption conditions and the validity period of the code. Codes cannot be redeemed after expiration.

  4. Each redemption code can only be used once. The same game cannot use redemption codes of the same type more than once.
